Exercícios: JS Responsivo e Acessível

1. Preferência de Movimento

Esta caixa deve parar se o usuário preferir "movimento reduzido". (dica: veja a aula 10)

2. Mude a pagina para um tema escuro

O navegador está em modo claro, pegue a cor das div e mude elas para que fiquem escuras e com texto claro ao apertar o botão. caso aperte denovo o botão ele ele deve retornar para o modo claro

3. Menu Mobile (Largura da Tela)

Em telas pequenas (menor que 600px), o botão menu ('btn-menu') deve começar escondido. (use 'window.innerWidth' para puxar o tamanho da tela de forma numerica)

4. Botão de Alto Contraste

Alguns usuários precisam de contraste máximo.

5. Aumentar Fonte (Zoom Manual)

Este texto precisa ser legível para todos, então caso clique no botão aumente o tamanho da fonte em +2 (exemplo font-size 12 vai para font-size 14)
