@media

Regra, introduzida em CSS3, possibilitou definir regras de estilo diferentes para diferentes tipos de média.

As consultas de mídias podem ser usadas para verificar muitas coisas, como:

  • Largua e altura de viewport
  • Orientaçãi da viewport (paisagem ou retrato)
  • Resolução

Usando consulta de mídias são uma técnica popular para oferecer um estilo personalizado folha para desktops, laptops, tablets e smartphones.
